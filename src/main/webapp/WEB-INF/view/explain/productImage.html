商品图片接口说明

  • POST：/api/productImage/add

    描述：添加商品图片接口

    参数: file(产品图片，不为空) ，product_id(外键，指向产品，不为空)

    附件：
    外键(指向产品的id)：

  • GET：/api/productImage/list

    描述: 查看首页商品图片列表

    参数：page_num(默认为第一页，可为空)，page_size(默认为10条数据，可为空)

  • POST：/api/productImage/update

    描述: 根据id修改商品图片参数

    参数: id(主键，不为空)，file(文件，为空)，product_id(外键，指向产品表，不为空)

    附件：
    主键：
    外键(指向产品的id)：

  • POST：/api/productImage/delete

    描述: 根据传入批量id删除对应的商品图片

    参数: id

    主键：

  • POST：/api/productImage/listByProductId/{product_id}

    描述: 根据传入product_id显示对应的商品图片

    参数: product_id(外键，指向产品表的id，不为空)

  • POST：/api/productImage/listById/{id}

    描述: 根据传入id查询对应的商品图片

    参数: id(主键，不为空)

  • GET：/api/productImage/selectByProductId

    描述: 通过product_id查看商品所有图片

    参数：id
