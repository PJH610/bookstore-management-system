• POST：/api/product/add

    描述：添加商品接口

    参数: name(名字，不为空) ，subtitle(小标题，不为空)，original_price(原价格，不为空)，promote_price(优惠价格，不为空)， stock(库存，不为空)，category_id(外键,指向用户表)

  • GET：/api/product/selectByCategoryId/{id}

    描述: 通过分类的id查看首页商品列表

    参数：id(主键(分类表id)，不为空)，page_num(默认为第一页，可为空)，page_size(默认为10条数据，可为空)

  • POST：/api/product/update

    描述: 根据id修改内容参数

    参数: id(主键，不为空)，subtitle(小标题，不为空)，original_price(原价格，不为空)，promote_price(优惠价格，不为空) stock(库存，不为空)，category_id(外键,指向用户表，不为空)

  • POST：/api/product/delete

    描述: 根据传入批量id删除对应的商品

    参数: id

  • GET：api/product/selectFindProductImg

    描述: 通过商品的id查看商品的所有信息和图片

    参数：id

  • POST：/api/product/selectByName

    描述: 根据商品名模糊查询

    参数: name(商品名不为空)，page_num(默认为第一页，可为空)，page_size(默认为10条数据，可为空)
