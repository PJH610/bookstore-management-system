订单管理

  • GET：/api/order/selectByOrderId

    • 描述：根据用户id查询订单

    参数：id

  • GET：//api/order/select

    • 描述：根据id查询订单和订单项

    参数：id

  • GET：/api/order/selectById

    • 描述：根据订单查询用户名

    参数：id

  • GET：/api/order/selectlist

    • 描述：查询订单查询用户名

    参数：id

  • POST：/api/order/delivery

    • 描述：插入订单

    参数：user_id address(收货人) receivre(地址) mobile(电话) user_message(用户描述) order_code(订单号) create_date(创建时间)

  • POST：/api/order/delivery

    • 描述：发货

    参数：id delivery_date

    参数：id

  • POST：/api/order/confirm

    • 描述：收货

    参数：id confirm_date

  • POST：/api/order/cancel

    • 描述：取消订单

    参数：id
