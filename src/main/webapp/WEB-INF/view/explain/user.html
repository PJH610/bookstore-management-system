用户文档

  • POST：/api/user/login

    描述：登录

    参数：mobile_phone (手机号，不为空)，password (密码，不为空)

  • POST：/api/user/register

    描述：注册

    参数：mobile_phone (手机号，不为空)，password (密码，不为空)，email (email，不为空)

  • POST：/api/user/alterPw

    描述：修改密码

    参数：id (主键，不为空)，password (密码，不为空)

  • GET：/api/user/list

    描述：查询全部用户

    参数：page_num (默认值为1，可为空)，page_size (默认值为10，可为空)

  • GET：/api/user/show

    描述：显示用户

    参数：id (主键，不为空）

  • 接口：/api/user/logout

    描述：注销登录

  • POST：/api/user/delete

    描述：删除用户

    参数：id (主键，不为空）
